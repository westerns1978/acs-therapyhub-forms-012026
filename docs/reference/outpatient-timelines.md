# Outpatient Documentation Timelines — Quick Reference

> **Educational reference only.** This explains Missouri's outpatient documentation deadlines. It does **not** show any specific client's deadlines or compliance status — the system tracks those from each client's actual record and surfaces them on the dashboard.

Missouri outpatient SUD treatment (9 CSR 30-3.195) runs on a set of recurring documentation deadlines. Missing them is a real audit risk, so the system is designed to track and surface them — but it helps to know the underlying clock.

---

## The deadlines

**Informed consent — at intake, renewed annually**
A written informed consent must exist at intake and be renewed at least every **365 days.** Progress notes and billing depend on a current, signed consent being on file.

**Initial goal — at intake**
At least one preliminary treatment target must be documented **at the time of intake** before a client moves to active status.

**Treatment plan — within the first 3 visits**
A comprehensive individualized treatment plan must be finalized within the **first three outpatient visits.** Plan for it early — the fourth visit depends on it.

**Treatment plan review — every 90 days**
The plan must be formally reviewed and updated at least **every 90 days** from when it was executed. A documented review keeps charting current.

**Treatment plan rewrite — annually**
The plan must be **rewritten in full at least every 365 days** of active care, not just reviewed.

**Sign-off — LMHP (or routed)**
Treatment plans are signed by an LMHP. PLPC/QAP may draft, but an LMHP co-signs.

**Diagnoses — DSM-5-TR only**
All diagnoses use current DSM-5-TR codes and specifiers, rendered by (or routed to) an LMHP.

---

## How to think about it

The pattern is a cascade: **consent → initial goal → treatment plan (by visit 3) → 90-day reviews → annual rewrite**, with consent itself renewing annually alongside. The system watches these clocks and flags what's coming due, so the deadlines work *for* the clinician rather than being something to track on paper.
